# Release 1.1.0

**Released:** December 9, 2023

This release adds a home menu, credits screen, and expands platform support.

<!-- truncate -->

## What's New

### Home Menu
Added a home menu with improved navigation and user experience.

### Credits Screen
Added a credits screen to acknowledge contributors and the project.

### Platform Support
- Added NetBSD installation instructions
- Added Arch Linux installation directions

### Docker Improvements
- Created a much smaller final Docker image for better distribution

### Bug Fixes
- Fixed history display on line 8
- Fixed issue where selecting a chess piece with unauthorized moves was not properly prevented
- Added "escape" event to helper menu

### Documentation
- Added contributing guidelines

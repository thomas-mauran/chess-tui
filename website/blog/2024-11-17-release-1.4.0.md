# Release 1.4.0

**Released:** November 17, 2024

This release includes an important rule fix and a new board rotation feature.

<!-- truncate -->

## What's New

### Turn Board Feature
You can now rotate the board to view it from either player's perspective, making it easier to play from any angle.

### Bug Fixes
- Fixed threefold repetition detection to properly handle draw scenarios

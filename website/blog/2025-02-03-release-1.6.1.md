# Release 1.6.1

**Released:** February 3, 2025

This release focuses on improving logging capabilities and fixing multiplayer connection issues.

<!-- truncate -->

## What's New

### Configurable Logging System
Added a comprehensive, configurable logging system with full documentation. This makes debugging and monitoring much easier.

### Multiplayer Connection Improvements
- Fixed connection issues with improved socket handling
- Enhanced logging for multiplayer connections to help diagnose issues

### History Display Fix
Corrected the moves history display to show moves accurately.

### Bug Fixes
- Fixed ESC key bug when playing against a bot

# Homebrew

**Chess-tui** can be installed with Homebrew on macOS and Linux.

## Installation

```bash
brew install thomas-mauran/tap/chess-tui
```

**Then run the game with:**

```bash
chess-tui
```

If you already have an older version installed, you can upgrade it with:

```bash
brew upgrade thomas-mauran/tap/chess-tui
```

For other package managers and installation methods, see the [Installation Guide](/docs/Installation/Packaging%20status).

# Arch Linux

**Chess-tui** can be directly built from the Arch Linux package manager.


## Installation

```bash
pacman -S chess-tui
```

This package is available on the [official Arch Linux repositories](https://archlinux.org/packages/extra/x86_64/chess-tui/).
